# FN-Commerce

* Components: 

   For the user:
    - A Landing Page with supplier of a product.
    - A page to show info and products of the supplier.
    - A page for the user to see his favorite suppliers.
    
    For the Supplier:
    - A profile page.
    - A page to manage the request.
    
* Functionalites: 
    - Sign in & Sign up.
    - A database.
    - Scheduling.


Stack: MERN
