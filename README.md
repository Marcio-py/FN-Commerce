## Technologies:

- HTML/CSS/JS
- Styled components
- ReactJS
- Vite

## How to start

The project was bootstraped with vite so

```
   # Clone it
       git clone https://github.com/Marcio-py/FN-Commerce.git

   # Go in the file
       cd FN-commerce

   # Install de dependencies
       npm i

   # Execute the code
       npm run dev

```
